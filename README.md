# Webpack

Latihan menggunakan [`webpack`](http://webpack.github.io/doc/) compiler.


### Clone

``` bash
$ git clone https://github.com/javastudio/webpack-lesson-1.git
$ cd webpack-lesson-1
```

### Install Dependencies

``` bash
$ npm install
```

### Build / Compile

``` bash
$ npm start
```

Selesai.

Buka file `index.html` di browser.
